#  Intelligent Student Feedback Analyzer

A comprehensive full-stack platform designed to bridge the gap between student feedback and faculty improvements. Using **AI-driven Sentiment Analysis**, the system categorizes suggestions to help administrators take meaningful action.

---

##  Key Features

###  Student Portal
* **Onboarding & Profile:** Captures Student ID, Dept, Year, and Sem upon first login.
* **Multi-Category Feedback:** Submit evaluations for **Academics, Training, Skills, and Events**.
* **Detailed Submissions:** Log Faculty Name, Course code, Rating, and Suggestions.
* **Feedback History:** Track status from **Pending** to **Reviewed** and view full details of past submissions.

###  Admin Dashboard
* **Analytics Overview:** Real-time metrics for Total Feedback, Pending Tasks, and Overall Ratings.
* **Sentiment Analysis:** Analyzes student suggestions to identify emotional tone and urgency.
* **Advanced Management:** Filter submissions by Dept, Year, Type, and Course.
* **Review System:** Direct response mechanism. Once an admin replies, the student's status updates automatically.
* **Advanced Insights:** Displays "Most Appreciated" and "Most Criticized" areas based on historical data.

###  Faculty Dashboard
* **Direct Feedback Loop:** Admin-to-Faculty communication channel for specific suggestions.
* **Performance Tracking:** Personalized insights for faculty to improve teaching methodologies.
